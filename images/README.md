# Images

Drop your real images here using these **exact filenames** and they'll appear automatically
(no code changes needed). Until a file exists, the site shows a polished gradient placeholder.

## Required

| Filename | Where it shows | Recommended size |
|---|---|---|
| `headshot.jpg` | Hero portrait + About section | 800×1000 (portrait, 4:5) |
| `pocket-defender.png` | Featured project banner | 1200×900 |

## Project screenshots (16:10 looks best)

| Filename | Project |
|---|---|
| `budgerton.png` | Budgerton |
| `tooele-county-resources.png` | Tooele County Resources |
| `blackrock-law.png` | Blackrock Law |
| `ward-app.png` | Ward App |
| `littlefield-legal.png` | Littlefield Legal |
| `littlefield-accounting.png` | Littlefield Accounting |

Tip: `.png` is expected for project shots and `.jpg` for the headshot. If you prefer a
different extension, update the matching `src` in `index.html` / `script.js`.
